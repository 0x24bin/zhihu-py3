==========
安装和测试
==========

直接安装
========
依赖于beautifulsoup4、requests、html2text，会自动安装。

..  code-block:: bash

    git clone https://github.com/7sDream/zhihu-py3.git
    cd zhihu-py3
    ./setup.py install
